This footer is used for all screens (Mobile, Tablet, Desktop) and is responsive while satisfying color contrast ration for accessibility.

Accessibility Considerations

For Footer:
Responsive bahavior of the footer. Used the "footer" element to enclose the footer content. This helps screen readers understand that this is the footer of the page. Ensured that links in the footer are descriptive by providing alternative text for link element and icons.
