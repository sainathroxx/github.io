About

This site serves as the documentation for the design system I created for my personal portfolio. It reflects my personality

Foundations

This section outlines the core design principles behind the system, including a clear typographic hierarchy, a carefully selected color palette, and different logo variations. These foundational choices help ensure visual consistency and express the brand's tone across all pages.

Components

Here, you’ll find reusable interface elements such as navigation bars, footers, and cards. Each component is designed to be functional, flexible, and visually consistent, making them easy to implement across different parts of the portfolio.

Page Patterns

This section showcases layout guidelines for multiple views of the homepage and a project page. These patterns help maintain structure and clarity while allowing flexibility in how content is presented.
